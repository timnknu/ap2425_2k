Приклади HTML

Розглянемо основні елементи та концепції, необхідні для створення базових веб-сторінок.

Структура документа

Кожен HTML-документ починається з декларації <!DOCTYPE html>, за якою слідує корінний елемент <html>. Всередині елемента <html> ми маємо секції <head> та <body>.

Секція Head

Секція <head> містить метадані про веб-сторінку, такі як <title>, скрипти та стилі.

Секція Body

Секція <body> містить видимий вміст веб-сторінки.

Форматування тексту

HTML надає різні елементи для форматування тексту, такі як:

  • Жирний: <b>Жирний Текст</b>
  • Курсив: <i>Курсивний Текст</i>
  • Підкреслений: <u>Підкреслений Текст</u>
  • Перекреслений: <s>Перекреслений Текст</s>
  • Надрядковий: <sup>Надрядковий Текст</sup>
  • Підрядковий: <sub>Підрядковий Текст</sub>
  • Виділений текст: <mark>Виділений Текст</mark>

Семантичні елементи

HTML надає семантичні елементи, які додають значення вмісту:

<blockquote> - це елемент цитати, який часто використовується для цитат.

Елемент <abbr> використовується для абревіатур, наприклад HTML.

Елемент <dfn> використовується для визначень, наприклад HTML.

Елемент <cite> використовується для цитування джерел, наприклад Мистецтво веб-дизайну.

Додаткові елементи

Ось деякі інші елементи, з якими ви можете зустрітися:

  • <div> та <span> для групування елементів
  • <pre> для попередньо форматованого тексту
  • <code> для вставки коду
  • <hr> для горизонтальних ліній
  • <br> для розривів рядків
  • HTML-коментарі: <!-- Це коментар -->
